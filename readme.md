# Jogo Rgb
# Sobre o projeto

<a href="https://gmrgb.netlify.app">https://gmrgb.netlify.app</a>

<p>Projeto feito para treinar minha lógica no JS.</p>
<p>Consiste em um projeto em que uma cor em formato rgb é dada e o usuário deve acertar qual cor é daquele rgb. As cores são geradas de maneira aleatória e não há limite de tentativas, pois é apenas para treinar o conceito de rgb.</p>
